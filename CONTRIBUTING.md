# Contributing to Financial Ratio Calculator

Thank you for considering contributing.

## How to contribute

1. **Fork** the repository on GitHub.
2. **Clone** your fork and create a branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** — keep commits focused and messages clear.
4. **Test** that the app runs:
   ```bash
   npm install
   npm run dev
   npm run build
   ```
5. **Push** to your fork and open a **Pull Request** against the default branch.

## Guidelines

- Follow the existing code style (React, Tailwind, JSX).
- For new features or ratios, open an **Issue** first to discuss.
- For bugs, include steps to reproduce and expected vs actual behavior.

## Code of conduct

Be respectful and constructive. This project is open to everyone.
